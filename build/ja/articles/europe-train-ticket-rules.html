欧州各国鉄道チケット利用規則完全ガイド

チケットのコードと記号をすべて理解し、罰金と混乱を回避

欧州鉄道チケット利用規則 | 5カ国比較

クイック結論：各国の規則は大きく異なる

  • 必須の改札：フランス（黄色）、イタリア（緑色）
  • 改札不要：ドイツ、スイス、スペイン
  • 最も重い罰金：イタリア（€50-200）、スイス（CHF 90）
  • 子供のポリシー：4-6歳以下は通常無料、4-14歳は半額
  • おすすめ：各国の公式アプリをダウンロードし、事前に規則を確認

フランス SNCF：最も複雑なシステム

チケットの表示規則

フランスの鉄道チケット（billetterie）は3つのタイプに分かれ、表示方法が完全に異なります：

1. 紙のチケット（Billet classique）

+----------------------------------+
| SNCF    TGV INOUI               |
|----------------------------------|
| PARIS GARE DE LYON              |
| LAUSANNE                        |
| 10:23  ->  13:20               |
| Car 12  Seat 42A (Window)      |
| Class: 2nd                       |
| PNR: ABC123                      |
| Fare: PREM'S (Non-refundable)   |
+----------------------------------+

重要な情報：

  • PNR：6桁の英数字コード、予約の唯一の識別子
  • Fare：運賃タイプ。PREM'Sは最安の変更・取消不可切符；LOISIRは変更・取消可のレジャー切符；PROはビジネス切符
  • Car/Seat：車両番号と座席番号。フランスのTGV座席はA/B（窓側/通路側）とC/D（向かい合わせのテーブル席）に分かれる

2. 電子チケット（E-billet）

SNCF Connectアプリまたはウェブサイトで購入、QRコードとして表示。注意：

  • 搭車前に必ず「アクティベート」（Composter）——アプリ内のボタンをタップ
  • スマートフォンの電池が切れた場合、駅の黄色い改札機（Composteur）で紙の証明を印刷できる
  • 車内検査時はQRコードを提示し、車掌がスキャンして確認

3. パス（Pass）

EurailまたはInterrailパスを使用する場合：

  • パスに旅行日を記入する必要がある（青または黒のインクで）
  • TGV/ICEの座席はSNCFアプリまたはウェブサイトで予約する必要がある（€10-20の予約料）
  • 座席を予約せずにTGVに乗車することは違反であり、€50+の追加切符が必要

重要な規則

  • 改札（Composter）：紙のチケットは搭車前に黄色い改札機で改札する必要がある（チケットを挿入し、「カチッ」という音を確認）。改札なしは無効とみなされ、€50の罰金
  • 変更：PREM'S切符は変更不可；LOISIR切符は出発前に変更可（差額を支払う必要あり）；出発後は変更不可
  • 子供：4-11歳は半額、4歳以下は無料（座席なし）

ドイツ DB：最も明確なシステム

チケットの表示規則

+----------------------------------+
| DB Bahn    ICE 957               |
|----------------------------------|
| Berlin Hbf  ->  München Hbf     |
| 08:30  ->  13:03                 |
|                                  |
| 1. Klasse:  Wg. 12, Pl. 42A      |
| 2. Klasse:  Wg. 22, Pl. 55B      |
|                                  |
| Sparpreis (Saver Fare)           |
| Zugbindung: Ja (Train bound)     |
+----------------------------------+

重要な情報：

  • Wg. (Wagen)：車両番号
  • Pl. (Platz)：座席番号。A/B/C/Dは座席タイプを示す
  • Zugbindung：特定の列車に縛り付けられているかどうか。Sparpreisは縛り付けられており、この列車のみ利用可；Flexpreisは縛り付けられておらず、当日の任意の列車で利用可
  • 1./2. Klasse：一等/二等座

重要な規則

  • 改札不要：ドイツの鉄道チケットは改札不要、そのまま乗車
  • 車内検査：車掌がランダムに検査、紙の切符またはアプリのQRコードを提示
  • 遅延保障：DBの列車が60分以上遅延した場合、Sparpreis切符は全額返金；120分以上遅延した場合、Flexpreis切符は全額返金
  • BahnCard：ドイツ鉄道の割引カード。BahnCard 25/50/100はそれぞれ25%/50%/100%割引。注意：BahnCardには年会費があるが、通常2-3回の長距離利用で元が取れる

イタリア Trenitalia：最も罰金に注意が必要

チケットの表示規則

+----------------------------------+
| Trenitalia    Frecciarossa 9574  |
|----------------------------------|
| Roma Termini  ->  Firenze SMN   |
| 09:30  ->  10:45                 |
|                                  |
| Carrozza 3, Posto 12A, Fine     |
| Classe: Standard                 |
|                                  |
| Tariffa: Super Economy           |
| Codice PNR: ABC123               |
+----------------------------------+

重要な情報：

  • Carrozza/Posto：車両/座席
  • Fine/Centrale：座席の位置。Fineは車両の後部（トイレに近い）、Centraleは中央部（より静か）
  • Tariffa：運賃タイプ。Base（変更・取消可）；Economy（変更不可、日付変更可）；Super Economy（変更・取消不可）

重要な規則

  • 必須の改札：紙のチケットは搭車前に緑色の改札機（Validatrice）で改札する必要がある。改札なしは€50-200の罰金
  • Regional列車：地域列車（R/RV）は座席指定なし、乗車後に自由に座る
  • Frecciarossa：座席を予約する必要があり、指定された座席に座る。予約なしでは乗車不可
  • 子供：4-12歳は半額、4歳以下は無料

スイス SBB：最も親切なシステム

チケットの表示規則

+----------------------------------+
| SBB CFF FFS    IC 1071           |
|----------------------------------|
| Zürich HB  ->  Genève            |
| 08:00  ->  11:02                 |
|                                  |
| 1. Klasse:  Wagen 1, Platz 21A   |
| 2. Klasse:  Wagen 5, Platz 42B   |
|                                  |
| Sparbillette (Saver Ticket)      |
| Halbtax-Abo: Ja                  |
+----------------------------------+

重要な情報：

  • Halbtax-Abo：半額カードを使用しているかどうか。スイスの住民はほぼ全員Halbtax（半額カード、年間CHF 185）を購入し、すべての切符が半額
  • GA Travelcard：スイスのパス（年間CHF 3860、二等座）、すべての公共交通機関を無制限に利用可
  • Supersaver：スーパー割引切符、事前購入で半額カードの25%価格まで安くなる

重要な規則

  • 改札不要：スイスの鉄道チケットは改札不要
  • 車内検査：ランダムに検査、頻度が高い（約80%の列車で検査）
  • 罰金：無券乗車はCHF 90+追加切符
  • 定時：スイス鉄道は定時性で有名。3分以内の遅延は遅延とみなされない；15分以上の遅延は一部返金を申請可

スペイン Renfe：最も特殊なシステム

チケットの表示規則

+----------------------------------+
| Renfe    AVE 03971               |
|----------------------------------|
| Madrid Puerta de Atocha         |
| Barcelona Sants                  |
| 08:30  ->  11:15                 |
|                                  |
| Coche 3, Plaza 12A, Ventana      |
| Clase: Turista                   |
|                                  |
| Tarifa: Promo                    |
| Localizador: ABC12345678         |
+----------------------------------+

重要な情報：

  • Coche/Plaza：車両/座席
  • Ventana/Pasillo：窓側/通路側
  • Tarifa：Promo（変更・取消不可）；Flexible（変更・取消可）；Premium（ビジネスクラス）
  • Localizador：10桁のロケーションコード、切符の受け取りと変更に使用

重要な規則

  • 保安検査：スペインの高速鉄道（AVE）は保安検査が必要、駅に30分前に到着
  • 座席タイプ：Turista（二等）；Turista Plus（二等+）；Preferente（一等）；Club（ビジネス）
  • 子供：4-13歳は半額、4歳以下は無料
  • Renfe+：会員プログラム、ポイントで切符に交換可

一般的な規則のまとめ

国 改札が必要 検査頻度 無券の罰金 子供のポリシー
フランス はい（黄色の機械） 高 €50+追加切符 4-11歳は半額
ドイツ いいえ 中 €60+追加切符 6-14歳は半額
イタリア はい（緑色の機械） 高 €50-200+追加切符 4-12歳は半額
スイス いいえ 非常に高 CHF 90+追加切符 6-16歳は半額

避けるべき落とし穴

  1. 常に事前に購入：欧州の鉄道チケットは2-3ヶ月前に購入すると50-70%安くなる
  2. 予約が必要か確認：高速列車（TGV、ICE、Frecciarossa、AVE）は予約が必須；地域列車は通常不要
  3. PNR/Localizadorを保存：これはあなたの予約証明、スクリーンショットを保存
  4. 公式アプリをダウンロード：SNCF Connect、DB Navigator、Trenitalia、SBB Mobile、Renfe
  5. 乗り換え時間に注意：欧州の駅での乗り換えは通常10-15分必要、大きな駅（例：パリ北駅）は20分必要
  6. 紙の切符を保存：電子チケットを使用しても、紙のバックアップを印刷しておくことをおすすめ
