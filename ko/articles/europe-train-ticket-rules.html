티켓의 모든 코드와 기호를 이해하고, 벌금과 혼란을 피하세요

유럽 철도 티켓 이용 규칙 | 5개국 비교

빠른 결론: 각국 규칙 차이가 큼

  • 반드시 찍기: 프랑스 (노란색), 이탈리아 (녹색)
  • 찍기 불필요: 독일, 스위스, 스페인
  • 벌금 가장 무거움: 이탈리아 (€50-200), 스위스 (CHF 90)
  • 어린이 정책: 4-6세 이하 보통 무료, 4-14세 반액
  • 추천: 각국 공식 앱 다운로드, 규칙 미리 확인

프랑스 SNCF: 가장 복잡한 시스템

티켓 표시 규칙

프랑스 철도 티켓은 세 가지 유형으로 표시 방식이 완전히 다릅니다:

1. 종이 티켓 (Billet classique)

+----------------------------------+
| SNCF    TGV INOUI               |
|----------------------------------|
| PARIS GARE DE LYON              |
| LAUSANNE                        |
| 10:23  ->  13:20               |
| Car 12  Seat 42A (Window)      |
| Class: 2nd                       |
| PNR: ABC123                      |
| Fare: PREM'S (Non-refundable)   |
+----------------------------------+

핵심 정보:

  • PNR: 6자리 영문자+숫자 코드, 예약의 유일한 식별자
  • Fare: 요금 유형. PREM'S는 가장 저렴한 환불/변경 불가 티켓; LOISIR는 환불/변경 가능한 여가 티켓; PRO는 비즈니스 티켓
  • Car/Seat: 객차 번호와 좌석 번호. 프랑스 TGV 좌석은 A/B (창가/복도)와 C/D (마주보는 테이블석)로 구분

2. 전자 티켓 (E-billet)

SNCF Connect 앱 또는 웹사이트로 구매, QR 코드로 표시. 주의사항:

  • 탑승 전 "활성화" (Composter) 필수 — 앱에서 버튼 클릭
  • 휴대폰 배터리 방전 시 역의 노란색 찍기 기계 (Composteur)에서 종이 증명서 출력 가능
  • 승차권 검사 시 QR 코드 제시, 차장이 스캔하여 검증

3. 패스 (Pass)

Eurail 또는 Interrail 패스 사용 시:

  • 패스에 여행 날짜를 반드시 기재 (파란색 또는 검은색 잉크)
  • TGV/ICE 예약은 SNCF 앱 또는 웹사이트에서 좌석 예약 필요 (€10-20 예약비)
  • 예약 없이 TGV 탑승은 위반, 벌금 €50+ 추가 요금

중요 규칙

  • 찍기 (Composter): 종이 티켓은 탑승 전 노란색 찍기 기계에서 찍어야 함 (티켓 삽입, "딸깍" 소리). 미찍기 시 무효, 벌금 €50
  • 변경: PREM'S 티켓 변경 불가; LOISIR 티켓 출발 전 변경 가능 (차액 보충); 출발 후 변경 불가
  • 어린이: 4-11세 반액, 4세 이하 무료 (좌석 미점유)

독일 DB: 가장 명확한 시스템

티켓 표시 규칙

+----------------------------------+
| DB Bahn    ICE 957               |
|----------------------------------|
| Berlin Hbf  ->  München Hbf     |
| 08:30  ->  13:03                 |
|                                  |
| 1. Klasse:  Wg. 12, Pl. 42A      |
| 2. Klasse:  Wg. 22, Pl. 55B      |
|                                  |
| Sparpreis (Saver Fare)           |
| Zugbindung: Ja (Train bound)     |
+----------------------------------+

핵심 정보:

  • Wg. (Wagen): 객차 번호
  • Pl. (Platz): 좌석 번호. A/B/C/D는 좌석 유형
  • Zugbindung: 특정 열차에 묶여 있는지 여부. Sparpreis는 묶여 있음, 이 열차만 탑승 가능; Flexpreis는 묶여 있지 않음, 당일 어떤 열차든 탑승 가능
  • 1./2. Klasse: 1등석/2등석

중요 규칙

  • 찍기 불필요: 독일 철도 티켓은 찍을 필요 없이 바로 탑승
  • 승차권 검사: 차장이 무작위로 검사, 종이 티켓 또는 앱 QR 코드 제시
  • 지연 보장: DB 열차 60분 이상 지연 시 Sparpreis 티켓 전액 환불; 120분 이상 지연 시 Flexpreis 티켓 전액 환불
  • BahnCard: 독일 철도 할인 카드. BahnCard 25/50/100은 각각 25%/50%/100% 할인. 주의: BahnCard는 연회비가 있지만, 보통 장거리 2-3회 탑승 시 본전 회수

이탈리아 Trenitalia: 벌금에 가장 주의해야 함

티켓 표시 규칙

+----------------------------------+
| Trenitalia    Frecciarossa 9574  |
|----------------------------------|
| Roma Termini  ->  Firenze SMN   |
| 09:30  ->  10:45                 |
|                                  |
| Carrozza 3, Posto 12A, Fine     |
| Classe: Standard                 |
|                                  |
| Tariffa: Super Economy           |
| Codice PNR: ABC123               |
+----------------------------------+

핵심 정보:

  • Carrozza/Posto: 객차/좌석
  • Fine/Centrale: 좌석 위치. Fine은 객차 후부 (화장실 근처), Centrale은 중부 (더 조용)
  • Tariffa: 요금 유형. Base (환불/변경 가능); Economy (환불 불가, 변경 가능); Super Economy (환불/변경 불가)

중요 규칙

  • 반드시 찍기: 종이 티켓은 탑승 전 녹색 찍기 기계 (Validatrice)에서 찍어야 함. 미찍기 시 벌금 €50-200
  • Regional 열차: 지역 열차 (R/RV)는 지정석 없음, 탑승 후 자유롭게 착석
  • Frecciarossa: 반드시 좌석 예약 필요, 지정석. 미예약 시 탑승 불가
  • 어린이: 4-12세 반액, 4세 이하 무료

스위스 SBB: 가장 친화적인 시스템

티켓 표시 규칙

+----------------------------------+
| SBB CFF FFS    IC 1071           |
|----------------------------------|
| Zürich HB  ->  Genève            |
| 08:00  ->  11:02                 |
|                                  |
| 1. Klasse:  Wagen 1, Platz 21A   |
| 2. Klasse:  Wagen 5, Platz 42B   |
|                                  |
| Sparbillette (Saver Ticket)      |
| Halbtax-Abo: Ja                  |
+----------------------------------+

핵심 정보:

  • Halbtax-Abo: 반값 카드 사용 여부. 스위스 거주자는 거의 모두 Halbtax (반값 카드, 연회비 CHF 185) 구매, 모든 티켓 반액
  • GA Travelcard: 스위스 패스 (연회비 CHF 3860, 2등석), 모든 대중교통 무제한 탑승
  • Supersaver: 슈퍼 할인 티켓, 미리 구매 시 반값 카드의 25% 가격까지 저렴

중요 규칙

  • 찍기 불필요: 스위스 철도 티켓은 찍을 필요 없음
  • 승차권 검사: 무작위 검사, 빈도 높음 (약 80% 열차 검사)
  • 벌금: 무효 탑승 시 벌금 CHF 90+ 추가 요금
  • 정시: 스위스 철도는 정시성으로 유명. 3분 이내 지연은 지연으로 간주하지 않음; 15분 이상 지연 시 부분 환불 신청 가능

스페인 Renfe: 가장 특별한 시스템

티켓 표시 규칙

+----------------------------------+
| Renfe    AVE 03971               |
|----------------------------------|
| Madrid Puerta de Atocha         |
| Barcelona Sants                  |
| 08:30  ->  11:15                 |
|                                  |
| Coche 3, Plaza 12A, Ventana      |
| Clase: Turista                   |
|                                  |
| Tarifa: Promo                    |
| Localizador: ABC12345678         |
+----------------------------------+

핵심 정보:

  • Coche/Plaza: 객차/좌석
  • Ventana/Pasillo: 창가/복도
  • Tarifa: Promo (환불/변경 불가); Flexible (환불/변경 가능); Premium (비즈니스 클래스)
  • Localizador: 10자리 위치 코드, 티켓 수령 및 변경에 사용

중요 규칙

  • 보안 검사: 스페인 고속철도 (AVE)는 보안 검사 필요, 출발 30분 전 역 도착
  • 좌석 유형: Turista (2등); Turista Plus (2등+); Preferente (1등); Club (비즈니스)
  • 어린이: 4-13세 반액, 4세 이하 무료
  • Renfe+: 회원 프로그램, 포인트로 티켓 교환 가능

통합 규칙 요약

국가 찍기 필요 검사 빈도 무효 벌금 어린이 정책
프랑스 예 (노란 기계) 높음 €50+ 추가 요금 4-11세 반액
독일 아니오 중간 €60+ 추가 요금 6-14세 반액
이탈리아 예 (녹색 기계) 높음 €50-200+ 추가 요금 4-12세 반액
스위스 아니오 매우 높음 CHF 90+ 추가 요금 6-16세 반액
스페인 아니오 중간 티켓 가격 100% 벌금 4-13세 반액

피해 방지 가이드

  1. 항상 미리 티켓 구매: 유럽 철도 티켓은 2-3개월 전 구매 시 50-70% 저렴
  2. 예약 필요 여부 확인: 고속열차 (TGV, ICE, Frecciarossa, AVE)는 반드시 예약 필요; 지역 열차는 보통 불필요
  3. PNR/Localizador 보관: 예약 증명이므로 스크린샷 저장
  4. 공식 앱 다운로드: SNCF Connect, DB Navigator, Trenitalia, SBB Mobile, Renfe
  5. 환승 시간 주의: 유럽 역 환승은 보통 10-15분 필요, 대형 역 (파리 북역)은 20분 필요
  6. 종이 티켓 보관: 전자 티켓 사용 시에도 종이 백업 출력 권장
