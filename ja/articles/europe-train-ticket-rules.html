欧州鉄道切符規則完全ガイド

切符のすべてのコードと記号を理解し、罰金と気まずい状況を避ける

European Train Ticket Rules | 5-Country Comparison

結論：規則は国によって大きく異なる

  • 必須確認: フランス (yellow), イタリア (green)
  • 確認不要: ドイツ, Switzerland, スペイン
  • 最高罰金: イタリア (€50–200), Switzerland (CHF 90)
  • 子供政策: Usually free under 4–6, half price 4–14
  • ヒント: Download each country’s official app and read the rules before you travel

フランスSNCF：最も複雑なシステム

チケットの表示方法

フランスの切符（ビレタリー）は3種類あり、それぞれ表示形式が全く異なります：

1. Paper ticket (Billet classique)

+----------------------------------+
| SNCF    TGV INOUI               |
|----------------------------------|
| PARIS GARE DE LYON              |
| LAUSANNE                        |
| 10:23  ->  13:20               |
| Car 12  Seat 42A (Window)      |
| Class: 2nd                       |
| PNR: ABC123                      |
| Fare: PREM'S (Non-refundable)   |
+----------------------------------+

重要情報：

  • PNR: 6文字の英数字コード、予約の固有参照番号
  • 運賃: 運賃タイプ. PREM'S is the cheapest non-refundable ticket; LOISIR is a refundable leisure ticket; PRO is a business ticket
  • 車両/座席: 車両番号と座席番号。フランスTGVの座席はA/B（窓側/通路側）とC/D（対面テーブル席）と表示されます

2. E-ticket (E-billet)

SNCF Connectアプリまたはウェブサイトで購入し、QRコードとして表示されます。注：

  • 乗車前に「アクティベート」（コンポスター）する必要があります—アプリのボタンをタップしてください
  • If your phone dies, you can print a paper voucher at the station’s yellow composting machine (Composteur)
  • 車掌にQRコードを見せてスキャンしてください

3. Rail pass (Pass)

EurailまたはInterrailパスを使用する場合：

  • パスに旅行日を記入する必要があります（青または黒のインクで）
  • SNCFアプリまたはウェブサイトでTGV/ICEの座席を予約する必要があります（予約料€10〜20）
  • 予約なしでTGVに乗車することは違反であり、罰金€50＋運賃となります

Important Rules

  • コンポスター： 紙の切符は乗車前に黄色い機械でコンポスターする必要があります（切符を入れて「カチッ」という音を聞く）。コンポスターしない場合は無賃乗車とみなされ、罰金€50となります
  • 乗り換え: PREM'S切符は変更不可；LOISIR切符は出発前に運賃差額で変更可能；出発後は変更不可
  • 子供： 4〜11歳は半額、4歳未満は無料（座席なし）

ドイツDB：最も明確なシステム

チケットの表示方法

+----------------------------------+
| DB Bahn    ICE 957               |
|----------------------------------|
| Berlin Hbf  ->  München Hbf     |
| 08:30  ->  13:03                 |
|                                  |
| 1. Klasse:  Wg. 12, Pl. 42A      |
| 2. Klasse:  Wg. 22, Pl. 55B      |
|                                  |
| Sparpreis (Saver Fare)           |
| Zugbindung: Ja (Train bound)     |
+----------------------------------+

重要情報：

  • Wg. (Wagen): 車両番号
  • Pl. (Platz): 座席番号。A/B/C/Dは座席タイプを示します
  • Zugbindung: 切符が特定の列車に縛られているかどうか。Sparpreisは列車縛り；Flexpreisは縛りなし、その日のどの列車でも有効
  • 1./2. Klasse: 1等車 / 2等車

Important Rules

  • 確認不要： ドイツの切符は捺印不要；そのまま乗車してください
  • 切符確認： 車掌がランダムに確認；紙の切符またはアプリのQRコードを見せてください
  • 遅延補償： DB列車が60分以上遅延した場合、Sparpreis切符は全額払い戻し可能；120分以上の場合、Flexpreis切符は全額払い戻し可能
  • Bahnカード： ドイツ鉄道割引カード。Bahnカード25/50/100は25%/50%/100%割引。注：Bahnカードには年会費がありますが、通常2〜3回の長距離旅行で元が取れます

イタリアTrenitalia：罰金に注意

チケットの表示方法

+----------------------------------+
| Trenitalia    Frecciarossa 9574  |
|----------------------------------|
| Roma Termini  ->  Firenze SMN   |
| 09:30  ->  10:45                 |
|                                  |
| Carrozza 3, Posto 12A, Fine     |
| Classe: Standard                 |
|                                  |
| Tariffa: Super Economy           |
| Codice PNR: ABC123               |
+----------------------------------+

重要情報：

  • Carrozza/Posto: 車両 / 座席
  • Fine/Centrale: 座席位置。Fineは車両の端近く（トイレ近く）；Centraleは中央（静か）
  • Tariffa: 運賃タイプ。Base（払い戻し可能）；Economy（払い戻し不可、日付変更可能）；Super Economy（払い戻し不可、変更不可）

Important Rules

  • 必須確認： 紙の切符は乗車前に緑の機械（Validatrice）で確認する必要があります。確認しない場合：罰金€50〜200
  • 地域列車： 地域列車（R/RV）は座席指定なし；乗車後どこにでも座れます
  • Frecciarossa： 座席予約必須、指定席。予約なしでの乗車はできません
  • 子供： 4〜12歳は半額、4歳未満は無料

スイスSBB：最も親切なシステム

チケットの表示方法

+----------------------------------+
| SBB CFF FFS    IC 1071           |
|----------------------------------|
| Zürich HB  ->  Genève            |
| 08:00  ->  11:02                 |
|                                  |
| 1. Klasse:  Wagen 1, Platz 21A   |
| 2. Klasse:  Wagen 5, Platz 42B   |
|                                  |
| Sparbillette (Saver Ticket)      |
| Halbtax-Abo: Ja                  |
+----------------------------------+

重要情報：

  • Halbtax-Abo: 半額カードを使用しているかどうか。ほとんどのスイス住民はHalbtax（年会費CHF185）を購入しており、すべての運賃が半額になります
  • GA Travelcard: スイス一般パス（2等車年間CHF3,860）、すべての公共交通機関で無制限に利用可能
  • Supersaver: スーパー割引切符、事前購入で半額運賃の25%まで安くなる場合があります

Important Rules

  • 確認不要： スイスの切符は捺印不要
  • 切符確認： ランダム確認、頻度が非常に高い（列車の約80%）
  • 罰金： 切符なしでの乗車：罰金CHF90＋運賃
  • 定時性： スイス鉄道は定時運行で有名。3分以内の遅延は遅延とみなされず；15分以上の遅延は一部払い戻しの対象

スペインRenfe：最もユニークなシステム

チケットの表示方法

+----------------------------------+
| Renfe    AVE 03971               |
|----------------------------------|
| Madrid Puerta de Atocha         |
| Barcelona Sants                  |
| 08:30  ->  11:15                 |
|                                  |
| Coche 3, Plaza 12A, Ventana      |
| Clase: Turista                   |
|                                  |
| Tarifa: Promo                    |
| Localizador: ABC12345678         |
+----------------------------------+

重要情報：

  • Coche/Plaza: 車両 / 座席
  • Ventana/Pasillo: 窓側 / 通路側
  • Tarifa: Promo（払い戻し不可）；Flexible（払い戻し可能）；Premium（ビジネスクラス）
  • Localizador: 10桁のロケーターコード、切符の取り出しと変更に使用

Important Rules

  • セキュリティチェック： スペインの高速列車（AVE）はセキュリティスクリーニングが必要；駅に30分前に到着してください
  • 座席タイプ： Turista（2等車）；Turista Plus（2等車+）；Preferente（1等車）；Club（ビジネス）
  • 子供： 4〜13歳は半額、4歳未満は無料
  • Renfe+： ロイヤリティプログラム、ポイントは切符と交換可能

一般規則のまとめ

国 確認必要 確認頻度 切符なしの罰金 子供政策
France はい（黄色い機械） 高 €50＋運賃 4〜11歳半額
Germany 不要 中 €60＋運賃 6〜14歳半額
Italy はい（緑の機械） 高 €50〜200＋運賃 4〜12歳半額
Switzerland 不要 非常に高 CHF90＋運賃 6〜16歳半額
Spain 不要 中 運賃の100%が罰金 4〜13歳半額

落とし穴回避ガイド

  1. 必ず早く購入： 2〜3ヶ月前に購入した欧州鉄道切符は50〜70%安くなる可能性があります
  2. 予約が必要か確認： 高速列車（TGV、ICE、Frecciarossa、AVE）は予約必須；地域列車は通常不要
  3. PNR/Localizadorを保存： これは予約の証明—スクリーンショットしてください
  4. 公式アプリをダウンロード： SNCF Connect、DB Navigator、Trenitalia、SBB Mobile、Renfe
  5. 乗り換え時間を確保： 欧州の駅の乗り換えは通常10〜15分必要；大きな駅（例：パリ北駅）は20分必要
  6. 紙のバックアップを保持： E切符でも、バックアップとして紙のコピーを印刷してください

Europe Train編集部による2025年1月の実地テストに基づく。規則は変更される場合があるため、常に各鉄道会社の最新規則を確認してください。
