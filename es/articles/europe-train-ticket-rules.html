Guía Completa de Reglas de Billetes de Tren por País en Europa

Comprende cada código y símbolo en tu billete para evitar multas y momentos incómodos

Reglas de Billetes de Tren en Europa | Comparación de 5 Países

Conclusión Rápida: Las Reglas Varían Mucho entre Países

  • Obligatorio validar: Francia (máquina amarilla), Italia (máquina verde)
  • No necesita validar: Alemania, Suiza, España
  • Multas más altas: Italia (€50-200), Suiza (CHF 90)
  • Política infantil: Menores de 4-6 años suelen viajar gratis, 4-14 años mitad de precio
  • Consejo: Descarga las apps oficiales de cada país y familiarízate con las reglas antes de viajar

Francia SNCF: El Sistema Más Complejo

Tipos de Billete

Los billetes franceses se dividen en tres tipos, cada uno con reglas completamente diferentes:

1. Billete en papel (Billet classique)

+----------------------------------+
| SNCF    TGV INOUI               |
|----------------------------------|
| PARIS GARE DE LYON              |
| LAUSANNE                        |
| 10:23  ->  13:20               |
| Car 12  Seat 42A (Window)      |
| Class: 2nd                       |
| PNR: ABC123                      |
| Fare: PREM'S (Non-refundable)   |
+----------------------------------+

Información clave:

  • PNR: Código alfanumérico de 6 dígitos, identificador único de la reserva
  • Fare: Tipo de tarifa. PREM'S es el más barato, no reembolsable; LOISIR permite cambios; PRO es business
  • Car/Seat: Número de vagón y asiento. En TGV francés, A/B son ventana/pasillo y C/D son asientos enfrentados

2. Billete electrónico (E-billet)

Comprado a través de SNCF Connect App o web, se muestra como código QR. Notas importantes:

  • Debes "activar" (Composter) antes de subir — pulsa el botón en la app
  • Si se te agota la batería, puedes imprimir un comprobante en la máquina amarilla de la estación
  • El revisor escanea el QR durante el control

3. Pase (Pass)

Usando Eurail o Interrail:

  • Debes escribir la fecha de viaje en el pase (tinta azul o negra)
  • Debes reservar asiento para TGV/ICE a través de SNCF App o web (tarifa de reserva €10-20)
  • Subir a un TGV sin reserva es una infracción, multa de €50 + billete de repuesto

Reglas Importantes

  • Validar (Composter): El billete en papel debe validarse en la máquina amarilla antes de subir (introducir el billete, escuchar el "clic"). Sin validar se considera sin billete, multa de €50
  • Cambios: PREM'S no permite cambios; LOISIR permite cambios antes de la salida pagando diferencia; después de la salida no se puede cambiar
  • Infantil: 4-11 años mitad de precio, menores de 4 años gratis (sin ocupar asiento)

Alemania DB: El Sistema Más Claro

Información del Billete

+----------------------------------+
| DB Bahn    ICE 957               |
|----------------------------------|
| Berlin Hbf  ->  München Hbf     |
| 08:30  ->  13:03                 |
|                                  |
| 1. Klasse:  Wg. 12, Pl. 42A      |
| 2. Klasse:  Wg. 22, Pl. 55B      |
|                                  |
| Sparpreis (Saver Fare)           |
| Zugbindung: Ja (Train bound)     |
+----------------------------------+

Información clave:

  • Wg. (Wagen): Número de vagón
  • Pl. (Platz): Número de asiento. A/B/C/D indican tipo de asiento
  • Zugbindung: Si está vinculado a un tren específico. Sparpreis es vinculado, solo válido para ese tren; Flexpreis no está vinculado, válido para cualquier tren ese día
  • 1./2. Klasse: Primera/segunda clase

Reglas Importantes

  • No necesita validar: Los billetes alemanes no requieren validación, sube directamente
  • Control: El revisor hace controles aleatorios, muestra el billete en papel o QR de la app
  • Garantía de puntualidad: Si el tren DB llega con más de 60 minutos de retraso, Sparpreis es reembolsable al 100%; con más de 120 minutos, Flexpreis también
  • BahnCard: Tarjeta de descuento de DB. BahnCard 25/50/100 ofrece 25%/50%/100% de descuento. Nota: tiene cuota anual, pero normalmente se amortiza en 2-3 viajes largos

Italia Trenitalia: Donde Más Hay que Tener Cuidado con las Multas

Información del Billete

+----------------------------------+
| Trenitalia    Frecciarossa 9574  |
|----------------------------------|
| Roma Termini  ->  Firenze SMN   |
| 09:30  ->  10:45                 |
|                                  |
| Carrozza 3, Posto 12A, Fine     |
| Classe: Standard                 |
|                                  |
| Tariffa: Super Economy           |
| Codice PNR: ABC123               |
+----------------------------------+

Información clave:

  • Carrozza/Posto: Vagón/asiento
  • Fine/Centrale: Posición del asiento. Fine es al final del vagón (cerca del baño), Centrale es el centro (más tranquilo)
  • Tariffa: Tipo de tarifa. Base (reembolsable); Economy (no reembolsable, permite cambio de fecha); Super Economy (no reembolsable ni modificable)

Reglas Importantes

  • Obligatorio validar: El billete en papel debe validarse en la máquina verde (Validatrice) antes de subir. Sin validar, multa de €50-200
  • Trenes Regional: Los trenes regionales (R/RV) no tienen asiento asignado, sube y siéntate donde haya sitio
  • Frecciarossa: Asiento obligatorio, asignado al comprar. Sin reserva no puedes subir
  • Infantil: 4-12 años mitad de precio, menores de 4 años gratis

Suiza SBB: El Sistema Más Amigable

Información del Billete

+----------------------------------+
| SBB CFF FFS    IC 1071           |
|----------------------------------|
| Zürich HB  ->  Genève            |
| 08:00  ->  11:02                 |
|                                  |
| 1. Klasse:  Wagen 1, Platz 21A   |
| 2. Klasse:  Wagen 5, Platz 42B   |
|                                  |
| Sparbillette (Saver Ticket)      |
| Halbtax-Abo: Ja                  |
+----------------------------------+

Información clave:

  • Halbtax-Abo: Si usa tarjeta de mitad de precio. Los residentes suizos casi todos compran Halbtax (cuota anual CHF 185), todos los billetes a mitad de precio
  • GA Travelcard: Pase suizo anual (CHF 3860, segunda clase), viajes ilimitados en todo transporte público
  • Supersaver: Billete súper ahorro, comprando con antelación puede costar hasta el 25% del precio con Halbtax

Reglas Importantes

  • No necesita validar: Los billetes suizos no requieren validación
  • Control: Controles aleatorios, muy frecuentes (unos 80% de los trenes)
  • Multa: Viajar sin billete, multa de CHF 90 + billete de repuesto
  • Puntualidad: Los ferrocarriles suizos son famosos por su puntualidad. Retraso de menos de 3 minutos no se considera retraso; más de 15 minutos permite solicitar reembolso parcial

España Renfe: El Sistema Más Especial

Información del Billete

+----------------------------------+
| Renfe    AVE 03971               |
|----------------------------------|
| Madrid Puerta de Atocha         |
| Barcelona Sants                  |
| 08:30  ->  11:15                 |
|                                  |
| Coche 3, Plaza 12A, Ventana      |
| Clase: Turista                   |
|                                  |
| Tarifa: Promo                    |
| Localizador: ABC12345678         |
+----------------------------------+

Información clave:

  • Coche/Plaza: Vagón/asiento
  • Ventana/Pasillo: Ventana/pasillo
  • Tarifa: Promo (no reembolsable ni modificable); Flexible (reembolsable); Premium (business)
  • Localizador: Código de localización de 10 dígitos, para recoger billete y cambios

Reglas Importantes

  • Seguridad: El AVE español requiere pasar por control de seguridad, llegar 30 minutos antes a la estación
  • Tipos de asiento: Turista (segunda); Turista Plus (segunda+); Preferente (primera); Club (business)
  • Infantil: 4-13 años mitad de precio, menores de 4 años gratis
  • Renfe+: Programa de membresía, puntos canjeables por billetes

Guía para Evitar Problemas

  1. Compra siempre con antelación: Los billetes de tren europeos comprados 2-3 meses antes pueden ahorrar 50-70%
  2. Confirma si necesitas reserva: Los trenes de alta velocidad (TGV, ICE, Frecciarossa, AVE) requieren reserva obligatoria; los regionales normalmente no
  3. Guarda el PNR/Localizador: Es tu comprobante de reserva, haz captura de pantalla
  4. Descarga la app oficial: SNCF Connect, DB Navigator, Trenitalia, SBB Mobile, Renfe
  5. Presta atención al tiempo de transbordo: En estaciones europeas, el transbordo suele necesitar 10-15 minutos; en estaciones grandes (como París Norte) 20 minutos
  6. Conserva el billete en papel: Aunque uses billete electrónico, se recomienda imprimir copia de respaldo
