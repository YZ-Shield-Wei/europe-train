欧洲各国火车票使用规则完全指南

看懂车票上的每一个代码和符号，避免罚款和尴尬

欧洲火车票使用规则 | 5国对比

快速结论：各国规则差异大

  • 必须打卡：法国（黄色）、意大利（绿色）
  • 无需打卡：德国、瑞士、西班牙
  • 罚款最重：意大利（€50-200）、瑞士（CHF 90）
  • 儿童政策：4-6岁以下通常免费，4-14岁半价
  • 建议：下载各国官方App，提前了解规则

法国 SNCF：最复杂的系统

车票显示规则

法国火车票（billetterie）分为三种类型，显示方式完全不同：

1. 纸质车票（Billet classique）

+----------------------------------+
| SNCF    TGV INOUI               |
|----------------------------------|
| PARIS GARE DE LYON              |
| LAUSANNE                        |
| 10:23  ->  13:20               |
| Car 12  Seat 42A (Window)      |
| Class: 2nd                       |
| PNR: ABC123                      |
| Fare: PREM'S (Non-refundable)   |
+----------------------------------+

关键信息：

  • PNR：6位字母数字编码，是预订的唯一标识
  • Fare：票价类型。PREM'S是最便宜的不可退改票；LOISIR是可退改的休闲票；PRO是商务票
  • Car/Seat：车厢号和座位号。法国TGV座位分为A/B（靠窗/过道）和C/D（面对面桌座）

2. 电子票（E-billet）

通过SNCF Connect App或网站购买，显示为二维码。注意：

  • 必须在登车前"激活"（Composter）——点击App中的按钮
  • 如果手机没电，可以在车站的黄色打卡机（Composteur）上打印纸质凭证
  • 查票时出示二维码，列车员扫描验证

3. 通票（Pass）

使用Eurail或Interrail通票时：

  • 必须在通票上填写旅行日期（用蓝色或黑色墨水）
  • 必须在SNCF App或网站上为TGV/ICE预订座位（€10-20预订费）
  • 未预订座位就上TGV属于违规，罚款€50+补票

重要规则

  • 打卡（Composter）：纸质票必须在登车前在黄色打卡机上打卡（插入车票，听到"咔嚓"声）。未打卡视为无票，罚款€50
  • 改签：PREM'S票不可改签；LOISIR票发车前可改签，需补差价；发车后不可改签
  • 儿童：4-11岁半价，4岁以下免费（不占座）

德国 DB：最清晰的系统

车票显示规则

+----------------------------------+
| DB Bahn    ICE 957               |
|----------------------------------|
| Berlin Hbf  ->  München Hbf     |
| 08:30  ->  13:03                 |
|                                  |
| 1. Klasse:  Wg. 12, Pl. 42A      |
| 2. Klasse:  Wg. 22, Pl. 55B      |
|                                  |
| Sparpreis (Saver Fare)           |
| Zugbindung: Ja (Train bound)     |
+----------------------------------+

关键信息：

  • Wg. (Wagen)：车厢号
  • Pl. (Platz)：座位号。A/B/C/D表示座位类型
  • Zugbindung：是否绑定特定车次。Sparpreis是绑定的，只能坐这班车；Flexpreis不绑定，当天任何车次都可
  • 1./2. Klasse：一等/二等座

重要规则

  • 不需要打卡：德国火车票不需要打卡，直接上车
  • 查票：列车员随机查票，出示纸质票或App二维码
  • 晚点保障：如果DB列车晚点60分钟以上，Sparpreis票可全额退款；晚点120分钟以上，Flexpreis票可全额退款
  • BahnCard：德国铁路折扣卡。BahnCard 25/50/100提供25%/50%/100%折扣。注意：BahnCard有年费，但通常坐2-3次长途就回本

意大利 Trenitalia：最需要注意罚款

车票显示规则

+----------------------------------+
| Trenitalia    Frecciarossa 9574  |
|----------------------------------|
| Roma Termini  ->  Firenze SMN   |
| 09:30  ->  10:45                 |
|                                  |
| Carrozza 3, Posto 12A, Fine     |
| Classe: Standard                 |
|                                  |
| Tariffa: Super Economy           |
| Codice PNR: ABC123               |
+----------------------------------+

关键信息：

  • Carrozza/Posto：车厢/座位
  • Fine/Centrale：座位位置。Fine是车厢尾部（靠近厕所），Centrale是中部（更安静）
  • Tariffa：票价类型。Base（可退改）；Economy（不可退，可改期）；Super Economy（不可退改）

重要规则

  • 必须打卡：纸质票必须在登车前在绿色打卡机（Validatrice）上打卡。未打卡罚款€50-200
  • Regional列车：区域列车（R/RV）不指定座位，上车后随意坐
  • Frecciarossa：必须预订座位，对号入座。未预订不能上车
  • 儿童：4-12岁半价，4岁以下免费

瑞士 SBB：最友好的系统

车票显示规则

+----------------------------------+
| SBB CFF FFS    IC 1071           |
|----------------------------------|
| Zürich HB  ->  Genève            |
| 08:00  ->  11:02                 |
|                                  |
| 1. Klasse:  Wagen 1, Platz 21A   |
| 2. Klasse:  Wagen 5, Platz 42B   |
|                                  |
| Sparbillette (Saver Ticket)      |
| Halbtax-Abo: Ja                  |
+----------------------------------+

关键信息：

  • Halbtax-Abo：是否使用半价卡。瑞士居民几乎都买Halbtax（半价卡，年费CHF 185），所有票半价
  • GA Travelcard：瑞士通票（年票CHF 3860，二等座），无限次乘坐所有公共交通
  • Supersaver：超级优惠票，提前购买可低至半价卡的25%价格

重要规则

  • 不需要打卡：瑞士火车票不需要打卡
  • 查票：随机查票，频率很高（约80%的列车会查）
  • 罚款：无票乘车罚款CHF 90+补票
  • 准点：瑞士铁路以准点著称。晚点3分钟以内不算晚点；晚点15分钟以上可申请部分退款

西班牙 Renfe：最特殊的系统

车票显示规则

+----------------------------------+
| Renfe    AVE 03971               |
|----------------------------------|
| Madrid Puerta de Atocha         |
| Barcelona Sants                  |
| 08:30  ->  11:15                 |
|                                  |
| Coche 3, Plaza 12A, Ventana      |
| Clase: Turista                   |
|                                  |
| Tarifa: Promo                    |
| Localizador: ABC12345678         |
+----------------------------------+

关键信息：

  • Coche/Plaza：车厢/座位
  • Ventana/Pasillo：靠窗/过道
  • Tarifa：Promo（不可退改）；Flexible（可退改）；Premium（商务舱）
  • Localizador：10位定位码，用于取票和改签

重要规则

  • 安检：西班牙高铁（AVE）需要安检，提前30分钟到达车站
  • 座位类型：Turista（二等）；Turista Plus（二等+）；Preferente（一等）；Club（商务）
  • 儿童：4-13岁半价，4岁以下免费
  • Renfe+：会员计划，积分可兑换车票

避坑指南

  1. 总是提前买票：欧洲火车票提前2-3个月购买可便宜50-70%
  2. 确认是否需要预订：高速列车（TGV、ICE、Frecciarossa、AVE）必须预订；区域列车通常不需要
  3. 保存PNR/Localizador：这是你的预订凭证，截图保存
  4. 下载官方App：SNCF Connect、DB Navigator、Trenitalia、SBB Mobile、Renfe
  5. 注意换乘时间：欧洲车站换乘通常需要10-15分钟，大站（如巴黎北站）需要20分钟
  6. 保留纸质票：即使使用电子票，也建议打印纸质备份
